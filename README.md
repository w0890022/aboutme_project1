## About Me Project #1
A personal profile website built as my first HTML and CSS project. This project showcases an introductory index page, a structured navigation bar, and styled content pages featuring images and personal interests.

## Features
*   **Responsive Navigation Bar:** Easy browsing between the home page and sub-pages.
*   **Multi-Page Layout:** Includes dedicated sections for personal details, favorite things, and travel experiences.
*   **Media Integration:** Styled images and custom layouts.
*   **Clean Styling:** Built entirely with custom CSS for structural layout and aesthetics.

## Project Structure
*   `index.html` - The main landing/home page.
*   `AboutMe.html` - Introduction and background details.
*   `favorite.html` - A collection of favorite hobbies, media, or interests.
*   `travel.html` - Highlights of places visited or bucket-list travel destinations.
*   `style.css` - The master stylesheet governing the look, feel, and typography of the site.
